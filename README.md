# Asteroids

## Description

fly ship around and shoot asteroids

## Installation

Instructions on how to run game locally:
```python
pip install pygame
python main.py

## Controls

WASD - Move ship
Space - Shoot
ESC - Pause

## Features

Ship movement
Asteroid spawning
Collision detection
